# UNOFOOD 微信小程序

“UNOFOOD（剩食拯救）”是一个对标 Too Good To Go 的微信小程序项目，连接消费者与餐饮商家，将临期或当日未售完的食物以“惊喜袋”形式优惠售出，减少浪费并提升商家收益。

## 项目目标
- 提供消费者端浏览、下单、到店核销的闭环体验。
- 提供商家端发布“惊喜袋”、库存与提货时间管理、订单核销能力。
- 提供基础运营后台能力（商家审核、订单查询、数据看板）。

详细产品需求参见：docs/PRD.md。

## 技术栈（MVP最适合选择）
- 前端：微信小程序原生（WXML/WXSS/TypeScript），轻量状态管理；腾讯位置服务、微信支付、订阅消息。
- 后端：微信云开发（TCB）云函数 + 云数据库 + 云存储 + 云调用（支付与消息）。
- 可扩展替代方案：NestJS + PostgreSQL/MySQL + Redis + API Gateway + Serverless/容器化。

## 开发与调试
- 使用微信开发者工具导入本项目目录 d:\\unofood。
- 根据 docs/PRD.md 的页面信息架构逐步实现页面与组件。
- 在代码中编写详细注释，添加必要的错误处理与日志记录（参考 .trae/rules/project_rules.md）。

## 运行说明
- 打开微信开发者工具，选择“导入项目”并指向 d:\\unofood。
- 设置 AppID（请使用你的小程序AppID），确保权限与支付能力在微信平台配置完成。
- 如使用云开发（推荐），在开发者工具中开通并绑定环境；后端云函数与数据库结构后续在实现阶段补充。
